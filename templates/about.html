Project Directive

SafeZard is an advanced interactive simulation designed to enhance workplace safety awareness in laboratory and facility environments. Utilizing immersive video scenarios and real-time decision-making protocols, the system trains users to identify, assess, and mitigate potential hazards effectively.

The platform employs a gamified learning approach, tracking user proficiency through analytics and providing immediate feedback on safety protocol adherence.

  • Platform: Web / Mobile
  • Framework: Flask / Python
  • Database: SQL / JSON
  • Status: Operational
  • License: MIT / Open Source
